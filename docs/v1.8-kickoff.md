# Sparkforge v1.8 — "Mirrorwound + The Codex" Sprint Kickoff

*Scoped July 12, 2026, from `v1.8-seeds.md` (Brandon's post-v1.7 device
notes) + kickoff decisions. Brandon's framing of the seeds: "road
detours with a couple extra miles, not a new destination." Pending the
Lyra creative pass (brief asks in Waiting-on). DoD gates between units
stay absolute; the cut-first list exists from day one.*

**Thesis:** The arena keeps growing (Mirrorwound, arena 4) and the game
learns to *remember* — a Codex Suite (synergies, cards, bestiary) turns
a run's discoveries into a permanent collection. The endgame loop earns
a new reward (Forge XP Coins), and the Remove Ads pitch finally explains
itself. v2.0's Mote gets its #1 prerequisite a version early.

## Decided — do not relitigate

- **Pairing: Mirrorwound (arena) + Codex Suite (system).** The codex IS
  a retention system — strong pairing. **Content>systems canon holds:**
  if the month strains, **the arena ships and the Codex Suite slips
  whole** (cut-first, Unit 5+). E1/B/C/D/E are the protected core and
  ship regardless.
- **E1 price fix = Unit 0.** The $2.99 label vs $3.99 charge is
  compliance-adjacent; first thing built. **Real fix: kill the
  hardcode** — pull the localized price string from the StoreKit product
  (IAPManager already fetches products). Never wrong in any storefront.
- **Card codex "discovered" = ever OFFERED** (revealed the moment a card
  appears in a selection, passed-over or not). Same semantics apply to
  synergies-seen. Rewards exploration; faster, livelier fill.
- **Forge XP Coins:** **flat value — the XP Boost (2x) ad does NOT double
  them** (fixed post-boss bonus lane, easy to reason about). **NO magnet
  pull** — walking to them is the point (health-orb canon). Ember-orange,
  large, spinning (xScale oscillation). Arena-wide scatter + despawn
  timer. Claims physics bit **`0x1 << 8`** → update Portfolio Registry
  ladder when built.
- **In-run "No Ads" affordance = upper-LEFT, above the buff badges.**
  Clear of the pause button — **the v1.7 pause hit-zone fix is not
  touched or shrunk.**
- **Bestiary is pulled forward from v2.0** (Brandon's call). **Reserve
  the hidden Mote `????` entry in the data model now** so the schema
  doesn't churn at v2.0 (`docs/mote-v2.0-handoff.md`).
- **Synergy unlock modal QUEUES, never stacks.** Multi-tier in one pick
  (Extra Pick chains) presents modals one after another.
- **Ashling shard protection = a scripted ~0.4–0.6s beat**, not an
  i-frame / invuln. Parent dies → shake/explode → THEN shards pop up,
  vulnerable once visible. The delay *is* the protection.

### Unit 1 — Arena tuning & shard fix (B1 + B2)
- **B1 Arena 1 pacing +smidge:** dial up slightly (candidates: spawn
  interval floor, melee thinning chance, ranged spawn chance/time —
  `GameConfig` only, small moves). Post-boss swarm stays untouched.
- **B2 Ashling shard spawn protection:** `spawnAshlingShards` currently

  beat between parent-death VFX and shard physics existing.
**DoD:** Arena 1 no longer "stand still and machine-gun"; Ashling
parent's death AoE can't insta-kill the trio; shards vulnerable once
visible. Both validated on device.

### Unit 2 — Forge XP Coins (C)
- New boss-death pickup alongside the existing XP-orb shower (KEEP the
  shower — it works brilliantly). Boss also erupts large spinning
  ember-orange coins scattered arena-wide.
- Physics bit `0x1 << 8`; despawn timer (tune); no magnet; collected →
  `pendingForgeXP` / `addForgeXP` pipeline; flat (no XP Boost doubling).
**DoD:** coins scatter arena-wide on boss death, read as distinct from
blue magnet / green health / small XP orbs; collect into forge XP;
60fps held during the post-boss swarm.

### Unit 3 — Arena selector locked-state cleanup (D)

  it; icon/text hierarchy tidied. Touches `refreshArenaSection`'s locked
  branch only.
**DoD:** locked arenas read clearly during a real unlock moment; no
other arena-selector regressions.

### Unit 4 — Remove Ads package: pitch & in-run (E2 + E3 + E4)

**DoD:** value-prop shows before purchase; in-run icon legible and clear
