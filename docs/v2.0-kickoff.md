# Sparkforge v2.0 — "The Arrival" Sprint Kickoff

The comfortable-walk-away milestone. v1.9 built the systems; v2.0 pours the
content through them — the fifth arena, the Mote, the first skins, Boss Mode,
and two new trees. This is the release Brandon can set down for 2–3 months and
feel good about. It is deliberately a **milestone delivered across several
weekly point releases** (v2.0.0 → v2.0.x), not a single monolithic drop —
DoD gates between units stay absolute; Brandon owns the interactive playthrough.

## Why this is the big one

v1.9 was systems-not-content on purpose: the tier engine, capstone framework
(zones / priority-target turrets / tier→event ladders), familiar + morph
systems, and the Forge Path are a **content-generation pipeline**. v2.0 is the
first full exercise of that pipeline — a themed arena, a scripted narrative
beat, cosmetic monetization surfaces, a permanent replay mode, and two trees
built almost entirely from banked mechanics. Systems-first in v1.9 = content
velocity now.

## The road (arc context — not all v2.0 scope)

- **v2.0 (this sprint):** Arena 5 + Mote + earned skin + skin framework +
  Boss Mode (with dials) + Growth tree + Panda secret tree + premium/panda skins.
- **Season 1 (unadvertised testbed):** the Growth/plant-themed biome + arenas
  *fulfill* the Growth tree seed. Not v2.0 — the tree ships now, the biome
  lands across later releases. We use it privately to rehearse the seasonal
  "seed → biome" grammar before real seasons are announced.
- **v2.1+ / later:** themed-zone geometry & hazards, procedural reshuffle,
  enemies-impair-Spark (Arena 6+), Familiar/Summoner set, Decay tree,
  breadth-synergy 7th tree, attack-speed laser capstone. **Marketing begins ~v2.1.**

## Decided — do not relitigate

- **Arena 5 is the trigger.** Its clear fires the Mote entrance, which fires the
  earned skin. These three are **ONE narrative/mechanical beat — never split.**
- **Arena 5 is 2× the size of Arena 4.** (Flagged up front, not mid-build.)
  Still a larger circle — themed geometry is v2.1+, not here.
- **Skin pricing canon:** **$4.99 is the baseline skin price.** Special skins are
  the exception — the **1-year-anniversary Mote promo skin is $9.99.** Every
  premium skin is additive IAP; every earned skin keeps the free path whole.
  **Skins are purely cosmetic — they NEVER gate or alter gameplay.** (Monetization
  canon holds: rewarded-only ads, additive IAP, no paywalls/energy/gates.)
- **Pandas are entering the arena.** The Panda secret tree AND panda skins both
  ship in v2.0. Not a want, a need.
- **Boss Mode ships with its dials in v2.0.** Engineer the tuning knobs early
  (compounding returns) — this is a permanent slot future releases add to freely.
- **Growth tree ships in v2.0.** The Growth *biome/arenas* do not (Season 1).
- **Non-v2.0 banked items stay post-v2.0.** No changes to that list.
- **Ship cadence:** build phased/gated, **ship as ONE v2.0 by default**; convert
  built-to-date into an early "pivot release" only if installs surge. (See below.)

## Unit sequence

Grouped into phases. Each phase ends at a natural, shippable point-release cut
(see "Ship cadence"). One unit at a time, DoD gate between, Brandon validates
on-device, one commit per unit at DoD-pass with explicit approval.

### Phase A — The Arrival (the coupled core → v2.0.0)

**Unit 1 — Skin framework + Earned Spark skin**
Reusable alt-appearance system for Spark (swap visual identity — flame detail,
eyes, particles — over the existing PlayerNode), a selection surface (title/menu),
persistence (new `sf_`/`sparkforge_` keys — never rename existing), and the
premium-IAP plumbing (StoreKit product + $4.99 baseline, price StoreKit-live like
Remove Ads). First content = the Earned Spark skin, behind an unlock flag (flipped
in Unit 3).
**DoD:** framework renders ≥1 alternate skin cleanly at 60fps; selection persists;
IAP plumbing wired (test product); earned skin present but locked; build clean, no new warnings.

**Unit 2 — Arena 5 (2× Arena 4) + enemies + boss + tier-aware cards**
The fifth arena at double Arena 4's size, its enemy set, the fifth boss (fully
qualified `BossNode.slagTitan`-style pattern), and tier-aware cards. Watch spawn
density + performance at 2× area.
**DoD:** Arena 5 reachable and clearable; 5th boss fights correctly; new cards
tier-aware; 60fps holds at 2× size with full spawns; no crashes; build clean.

**Unit 3 — The Mote entrance + earned-skin payoff**
Arena 5 clear → the scripted Mote one-shot ("Mote the Terror," the "wait, wtf"
beat) → Codex `????` slot reveal → the Earned Spark skin unlocks as the reward.
Wire the beat as one continuous moment.
**DoD:** clearing Arena 5 triggers the Mote sequence; the one-shot reads as
intentional (not a bug); Codex slot resolves; earned skin unlocks + is selectable;
Brandon's on-device sign-off on the *feel* of the beat.

*→ v2.0.0 ships here: Arena 5 + Mote + earned skin + skin framework.*

### Phase B — Boss Mode (the permanent slot → v2.0.1)

**Unit 4 — Boss Mode core**
A dedicated bosses-only mode: auto-registering boss pool (every future boss joins
for free), the menu slot, and the two default modes — **Sequential** (arena order:
1, 2, 3, …) and **Mixup** (random bosses spawn; count capped/adjustable).
**DoD:** both modes playable start→finish; pool auto-includes all 5 bosses; mixup
cap configurable; mode is sandbox-isolated (see guardrail); build clean.

**Unit 5 — Boss Mode dials + 5-boss tuning pass**
The challenge-run knobs: independently dial boss HP / ATK / DEF up **and** down
(buffs and nerfs), plus the mixup count. Default = tuning as we configure it. Then
tune all 5 bosses together as a *set* for succession play.
**DoD:** every dial works live and persists per-config; default preset intact; the
5 bosses feel fair in sequence; Brandon's on-device sign-off.

*→ v2.0.1 ships here: Boss Mode.*

### Phase C — Trees + skin content (→ v2.0.2, v2.0.3)

**Unit 6 — Growth tree (7th tree)**
The Growth tree wired onto the v1.9 capstone engine (Terra gates the pool, Tree is

**Unit 8 — Premium & Panda skins**
Reusing Unit 1's framework: Premium Spark (elevated earned Spark), Earned Panda

**DoD:** all three render + select + persist; premium skins purchasable; earned

*→ v2.0.2 (trees) and v2.0.3 (skins) ship here.*

**DoD:** the full v2.0 experience holds together; Brandon's comfortable-walk-away sign-off.

## Guardrails

- **Boss Mode is a sandbox — isolate its rewards.** Because dials can *nerf*
  bosses, a Boss Mode clear must NOT grant the same unlocks/progression as a
  legitimate Arena clear (else nerfed-boss runs cheese the earned skins/Codex).
  Boss Mode is for play, not progression.
- **Skins never touch gameplay.** Pure cosmetic. No stat, hitbox, or ability delta.
- **Panda secrecy is load-bearing.** The earned-Panda-skin unlock and any UI it
  touches must not reveal the tree's logic. `???` forever.
- **2× arena = perf risk.** Validate 60fps with full spawn density before Unit 2 passes.

## Ship cadence — DECIDED (Jul 21 2026)

**Build cadence and release cadence are decoupled.** We build the phases in order
as gated units (A→B→C→D), but by **default ship the whole thing as ONE v2.0
release** when the full phased pass is complete. Rationale: player base is tiny
and unmarketed (marketing waits for ~v2.1), v1.9 just landed, so there's no
pressure to drip-feed.

**The ripcord:** if installs surge before we're done, we **pivot** — release
whatever's built to that point as the cut, then continue building toward the
full v2.0. So each phase boundary (v2.0.0 / .1 / .2 / .3 above) is a *potential*
release point kept shippable, but not a scheduled one. Flat installs (expected) →
keep building to the full phased pass, ship once. Brandon walks away after it lands.
