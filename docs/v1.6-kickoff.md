# Sparkforge v1.6 — "The Second Arena" Sprint Kickoff

*Kickoff date: July 8, 2026. Planning thread: Claude Code (this repo's first session under the new workflow). Supersedes the abandoned v1.6 chat thread.*

**Thesis:** Make the first arena's promises true, then open the second door. Every card does what it says, the Slag Titan is reachable and killable, and Arena 2 gives veterans a reason to return.

- **Arena 1 gate drops the boss-kill requirement.** New gate: 500 total kills + survive 2:00. The boss-kill gate *concept* is retained for Arena 2+ (where boss kills are actually earnable from Arena 1's Titan).
- **Ghost cards get fixed, not cut.** Lyra co-designed them; honor the designs.
- **XP Boost ad unit ID in code is wrong.** Canonical (verified in AdMob console 7/8/26): `ca-app-pub-3734133983597932/4094068573`.
- **No interstitials, ever. No energy systems.** All new ad surface is opt-in rewarded with the free path whole.
- **Game Center is approved** for the portfolio — but ships in v1.8 (Daily Trial), not v1.6.
- Locked design canon per CLAUDE.md (joystick, orb colors, non-magnetized health orbs).
