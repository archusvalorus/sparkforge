# Cross-App Cross-Promotion — "Hallways" Note (roadmap seed)

*Brandon's seed, July 13 2026. NOT scoped yet — a note to implement "at some
point." Framing (his): we've built an integrated ecosystem — a beautiful
cathedral — now we connect the rooms with hallways.*

## The idea

A tiny, tasteful cross-promo surface inside Sparkforge that points players to
the studio's other apps — and, reciprocally, the same treatment in those apps
pointing back. Turn a portfolio of separate titles into a connected ecosystem.

## Concrete shape

- **Title-screen banner** — small, low-emphasis, out of the way of the core
  menu/codex/Remove-Ads band. Copy along the lines of:
  - *"More Titles from the Forge"*
  - *"Like this? Here's some of our other apps"*
- Tapping it → the studio's other App Store listings (or an in-app "more
  from us" panel).

## Notes / dependencies

- **Studio canon:** the app roster lives in the Notion **App Portfolio
  Registry** (Forgebound Labs) — that registry is the source of truth for
  which titles to feature and their store links.
- **Reciprocity:** to be real cross-promotion, the *other* apps need the same
  banner pointing back — this is a studio-wide pattern, not a Sparkforge-only
  feature.
- **Monetization canon holds:** this is discovery, not a paywall/gate. Keep it
  optional, quiet, and never in the field of play.
- Placement interacts with the v1.8 **title bottom-band** work (Remove Ads +
  codex/bestiary/synergies entry points) — if this lands near that era, design
  the band to accommodate one more low-key affordance.

## Status

Banked. Not scoped. Natural fit once the title bottom-band is settled and the
Portfolio Registry has stable store links for the sibling apps.
